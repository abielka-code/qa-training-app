# Tasks QA Demo (GitHub Pages)

Навчальний веб-сайт для дисципліни "Тестування програмних систем і комплексів".
Працює без збірки, без npm: лише HTML/CSS/JS.

## Як увімкнути GitHub Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main (root)
4. Save
5. Отримаєте посилання: https://<username>.github.io/<repo>/

## Керовані дефекти
У файлі `config.js` можна вмикати/вимикати дефекти.
Це корисно для контрольованого оцінювання.
